核心特性

专为数据安全合规场景设计，开箱即用

🔌
完全离线运行
无需联网，数据处理全程在本地完成。无追踪、无遥测、无第三方数据上报，隐私安全有保障。
🔍
21 种敏感类型检测
覆盖个人隐私、金融信息、设备网络、商业政务四大类别，自动识别手机号、身份证、银行卡、IP地址等。
🎭
幂等假数据生成
基于种子随机数生成器，相同原始值始终生成相同假数据，保证数据集内部一致性，适合开发测试场景。
🔑
AES-256-GCM 可逆加密
使用 PBKDF2（10万次迭代）派生密钥，随机 IV + 认证标签，保管密钥即可还原原始数据。
📊
实时统计面板
处理后即时显示按类别统计的检测数量，保留最近 20 条处理历史，支持一键回看对比。
📁
多格式支持
支持 Excel（含多工作表）、Word、CSV、JSON、XML、TXT，拖拽即可处理，无需额外配置。
