### 🛡️ 脱敏策略

| 策略 | 效果示例 | 适用场景 |
|------|---------|---------|
| **🔒 掩码遮蔽** | `138****5678` | 数据展示、报告分享 |
| **#️⃣ 哈希脱敏** | `a1b2c3d4...` (SHA-256) | 数据比对、不可逆场景 |
| **🎭 假数据替换** | `13812345678` → `19988888888` | 开发测试、数据交付 |
| **🔑 可逆加密** | AES-256-GCM 加密 | 需要还原的场景 |

> **假数据特性**：基于幂等随机数生成器，相同原始值始终生成相同假数据，保证数据一致性。

## 🚀 快速开始

### 📦 下载安装

**方式一：便携版 ZIP（推荐）**

前往 [Releases 页面](https://github.com/lordcvader2/DataMaskStudio/releases/latest) 下载最新版本：

```
DataMaskStudio_v1.1.0_portable.zip  (约 163 MB)
```

解压到任意目录，双击 `DataMaskStudio.exe` 即可运行，**无需安装任何运行时**。

---

### 📖 使用方法

```
1. 启动程序  →  双击 DataMaskStudio.exe
2. 导入文件  →  拖拽文件到窗口，或点击选择文件
3. 配置选项  →  选择脱敏强度（轻/中/重）和策略（掩码/哈希/假数据/可逆）
4. 预览对比  →  左侧原始内容，右侧脱敏结果，敏感数据高亮标记
5. 保存结果  →  点击"保存文件"导出脱敏后的文件
```

## 🏗️ 技术架构

```
┌─────────────────────────────────────────────────────────┐
│                    UI 层 (React 18)                      │
│  拖拽/选择 → 配置选项 → 预览对比 → 统计面板 → 历史记录   │
└──────────────────────────┬──────────────────────────────┘
                           │ IPC (contextBridge)
┌──────────────────────────▼──────────────────────────────┐
│              Electron 主进程 (Node.js)                   │
│   文件读取 → 格式检测 → 内容解析 → 脱敏引擎              │
│                  ↓           ↓          ↓               │
│            xlsx 解析    mammoth     正则规则引擎          │
│            (Excel)      (Word)    (21种检测+4种策略)     │
│                                        ↓               │
│                              幂等假数据生成器             │
│                              AES-256-GCM 可逆加密        │
└─────────────────────────────────────────────────────────┘
```

### 核心模块

| 模块 | 技术 | 说明 |
|------|------|------|
| UI 框架 | React 18 + Ant Design 5 | 现代响应式界面，支持深色模式 |
| 构建工具 | Vite 6 + TypeScript 5.7 | 快速热更新，高类型安全 |
| Excel 解析 | xlsx (SheetJS) | 支持所有 Excel 格式变体 |
| Word 解析 | mammoth | 高保真 Word 文档提取 |
| 脱敏引擎 | 正则表达式 + 幂等PRNG | 21 种敏感类型，4 种策略 |
| 可逆加密 | AES-256-GCM (Node.js crypto) | PBKDF2 密钥派生，随机 IV |

## 📂 项目结构

```
DataMaskStudio/
├── electron/
│   ├── main.ts          # 主进程：文件解析 + 脱敏引擎（21种规则）
│   └── preload.ts       # 预加载脚本（IPC 桥接）
├── src/
│   ├── App.tsx          # React 主组件（统计面板、历史记录、预览对比）
│   ├── engine/          # 渲染进程脱敏引擎（与主进程同步）
│   │   ├── masker.ts    # 脱敏策略实现
│   │   ├── rules.ts     # 敏感信息检测规则
│   │   └── fakeDataGenerator.ts  # 幂等假数据生成器
│   ├── index.css        # 全局样式（Tailwind + 自定义变量）
│   └── renderer.d.ts    # TypeScript 类型声明
└── release/             # 构建输出
    └── win-unpacked/    # 绿色便携版目录
```

## 🔒 安全说明

| 特性 | 说明 |
|------|------|
| 🔌 完全离线 | 无需互联网连接，数据处理全程在本地完成 |
| 🚫 不收集信息 | 无追踪、无遥测、无第三方数据上报 |
