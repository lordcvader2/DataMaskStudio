# DataMask Studio

> 🔐 本地离线敏感数据脱敏工具 — 保护隐私，安心分享

一个轻量、高效的桌面应用程序，支持对任意格式的文件进行本地脱敏处理，**无需联网，数据不出本机**。

## ✨ 功能特性

### 支持的文件格式

| 格式 | 示例扩展名 | 备注 |
|------|-----------|------|
| Excel | `.xlsx` `.xls` | 含多工作表、合并单元格、公式等 |
| Word | `.docx` | 完整文档解析 |
| CSV | `.csv` | 自动识别逗号/分号分隔符 |
| JSON / JSONL | `.json` `.jsonl` | 含嵌套结构 |
| XML | `.xml` | 完整标签内容提取 |
| 纯文本 | `.txt` | 键值对、固定宽度等 |

### 敏感数据检测类型

| 类型 | 示例 | 说明 |
|------|------|------|
| 📱 手机号 | `13812345678` | 全号检测，支持所有号段 |
| 📧 邮箱 | `user@example.com` | 标准邮箱格式 |
| 🪪 身份证 | `110101199001011234` | 18位身份证号 |
| 💳 银行卡 | `6222021234567890` | 16-19位卡号 |
| 🌐 IP地址 | `192.168.1.1` | IPv4 全格式 |
| 💬 QQ号 | `1234567890` | 5-10位数字 |
| 🆔 护照号 | `E12345678` | 中英文护照格式 |
| 🔢 MAC地址 | `00:1A:2B:3C:4D:5E` | 标准MAC格式 |
| 🖥️ 微信号 | `wechat_id_xxx` | 含"微信号"关键词 |
| ☎️ 固定电话 | `010-12345678` | 固话格式 |

### 脱敏策略

| 策略 | 效果 | 适用场景 |
|------|------|----------|
| **掩码遮蔽** | 手机号 `138****5678`，身份证 `110101**********1234` | 数据展示、报告分享 |
| **哈希脱敏** | `1a2b3c4d` | 数据比对、不还原原始值 |
| **假数据替换** | `13812345678` → `19988888888` | 开发测试、数据交付 |
| **可逆加密** | `base64(reverse())` | 需还原的场景 |

### 脱敏强度

- **轻度**：保留更多原始信息（手机号保留前3后4位）
- **中度**：平衡保护与可读性（手机号保留前3后4位，更多中间位遮蔽）
- **重度**：最大程度保护（手机号仅保留前3位，其余全遮蔽）

## 🚀 快速开始

### 下载安装

**方式一：直接下载（推荐）**
```
下载 release 目录下的 DataMaskStudio.exe，双击运行即可
```

**方式二：便携版 ZIP**
```bash
下载 DataMaskStudio_v1.0.3_portable.zip
解压到任意目录
运行 DataMaskStudio.exe
```

### 使用方法

1. **启动程序** — 双击 `DataMaskStudio.exe`
2. **拖拽文件** — 将待脱敏文件拖入程序窗口，或点击选择文件
3. **配置选项** — 选择脱敏强度（轻度/中度/重度）和策略（掩码/哈希/假数据/可逆）
4. **预览对比** — 查看原始内容与脱敏内容对照
5. **保存结果** — 点击"保存文件"下载脱敏后的文件

## 🏗️ 技术架构

```
┌─────────────────────────────────────────────────────┐
│                    UI 层 (React)                     │
│   拖拽/选择 → 配置选项 → 预览对比 → 保存结果         │
└──────────────────────┬──────────────────────────────┘
                       │ IPC (contextBridge)
┌──────────────────────▼──────────────────────────────┐
│              Electron 主进程 (Node.js)             │
│   文件读取 → 格式检测 → 内容解析 → 脱敏处理          │
│                  ↓         ↓         ↓             │
│              xlsx 解析   mammoth   正则规则引擎      │
│              (Excel)   (Word)     (检测+脱敏)      │
└─────────────────────────────────────────────────────┘
```

### 核心模块

| 模块 | 技术 | 说明 |
|------|------|------|
| UI 框架 | React 18 + Ant Design 5 | 现代响应式界面 |
| 构建工具 | Vite + TypeScript | 快速热更新，高类型安全 |
| Excel 解析 | xlsx (SheetJS) | 支持所有 Excel 格式变体 |
| Word 解析 | mammoth | 高保真 Word 文档提取 |
| 脱敏规则 | 正则表达式引擎 | 10+ 种敏感类型全覆盖 |

## 🔒 安全说明

- **完全离线**：无需互联网连接，数据处理全程在本地完成
- **不收集信息**：无追踪、无遥测、无第三方依赖
- **内存安全**：敏感数据处理后即时释放，不在磁盘留下临时文件
- **开源透明**：代码完全可见，可自行审计安全性
